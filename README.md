
## Welcome to Fly-UI


### How to used

```
npm i Fly-UI
```

###文件夹说明
examples用来写组件相关文档和事例。
src目录里面只放一个main.js用来作为打包入口
packages 文件夹，我们组件的编写都在packages文件夹中进行


在main.js 中

```
import orange from 'orange_collect'
import 'orange_collect/lib/orange_collect.css'

Vue.use(orange)
```
